Voquill is built in the open. The source code is public, development happens transparently, and the community plays an active role in shaping the product.

## Discord

Join the [Voquill Discord server](https://discord.gg/5jXkDvdVdt) to chat with other users and the team behind Voquill. It's the best place to:

- Ask questions and get help
- Share feedback and feature requests
- Follow along with development updates
- Connect with other voice-to-text enthusiasts

## GitHub

Voquill's source code lives on [GitHub](https://github.com/josiahsrc/voquill). You can:

- Browse the code
- Report bugs and request features via issues
- Contribute pull requests

## Get in Touch

Have something that doesn't fit Discord or GitHub? Reach out at [hello@voquill.com](mailto:hello@voquill.com).
